# Tech stack

-   Framework: [Astro](https://astro.build/)
-   Deployment: [Netlify](https://netlify.com)
-   Styling: [TailwindCSS](https://tailwindcss.com/)

# Running locally

```bash
git clone https://github.com/tygrdotdev/tygr.dev
cd tygr.dev
pnpm install && pnpm dev
```
